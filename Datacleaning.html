DATA REDUCTION AND DATA CLEANING



Transforming the raw data in a useful and efficient format.The following steps are carried in Data Preprocessing

  • Handling Missing Values

    • Missing data is defined as the values or data that is not stored (or not present) for some variable/s in the given dataset.So that all algorithms doesn’t work properly with missing data and hence we have to handle it properly.In our dataset features such as ID, N_Days, Age, Edema and some others have missing values. All of this are handled here as a first step.

  • Outlier Removal


    • Before outlier removal



    Sometimes a dataset can contain extreme values that are outside the range of what is expected and unlike the other data (unusual event). The columns Bilirubin, Cholesterol, Albumin, Copper, Alk_Phos, SGOT, Tryglicerides, Platelets, Prothrombin contains outliers and which is replaced by flooring and capping and is shown in the below figure.

    After outlier removal




  • Feature Reduction


    • Correlation Heatmap



    Correlation between variables is also found with the help of Heatmap and since no variables are highly corelated with each other every variables should be included in model and can affect the dependent variable Stage.But here Some of the columns(ID, N_Days, Status) doesn’t contribute or fit to the feature prediction. So these variables are removed




  • Encoding

    • Encoding is a technique of converting categorical variables into numerical values so that it could be easily fitted to a machine learning model.Here we are using Label Encoding: This technique will be used only for Ordinal categories. Ranks are provided based on the importance of the category. Categorical data (Status, Drug, Sex, Ascites, Hepatomegaly, Spiders and Edema ) is replaced with Integer.


  • Handling Skewness


    • Before Removing Skewness



    A data is called as skewed when curve appears distorted or skewed either to the left or to the right, in a statistical distribution. Log transformation is used to remove the right-skewness of numerical data.
